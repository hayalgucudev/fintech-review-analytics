# Project Structure

## data/
Contains raw, processed, and output datasets.

### raw/
Original scraped review dataset.

### processed/
Cleaned and sentiment-enriched datasets.

### outputs/
Generated analytical summaries and keyword extraction outputs.

---

## notebooks/
Contains exploratory and reporting notebooks.

1. 01_data_collection.ipynb
2. 02_sentiment_analysis.ipynb
3. 03_thematic_analysis.ipynb
4. 04_business_insights.ipynb

---

## scripts/
Contains reusable Python scripts for scraping, preprocessing, analysis, and database loading.

---

## sql/
Contains PostgreSQL schema and verification queries.

---

## tests/
Contains unit tests for preprocessing and thematic analysis functions.
